   答案是：当前代码中没有地方主动设置动画名！
   
   rouletteAnim 字段的值来源只有两个：

3. NBT 读取（从存档加载）- EntityMaid.java:1430

4. 网络包同步（服务端 → 客户端）- SyncYsmMaidDataPackage.java:60

- 如果没有外部代码调用，永远是 "empty"

   当前状态：
- 女仆模组只提供了同步机制和API 接口

- 没有主动设置动画名的代码

- 需要外部模组或脚本调用 API 来触发动画
  
  如果你想让女仆播放 YSM 动画：
1. 编写 KubeJS 脚本调用 maid.playRouletteAnim("动画名")
